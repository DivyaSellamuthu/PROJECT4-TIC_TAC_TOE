Tic Tac Toe (3x3 Grid)
📌 Project Description

* This project is a simple Tic Tac Toe game implemented using a 3×3 grid.
* It is a two-player console-based game where players take turns marking spaces on the grid with X and O.
* The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

🎮 Game Rules

-> The game is played on a 3×3 grid.
-> Player 1 uses X and Player 2 uses O.
-> Players take turns entering their move.
-> A player wins if they place three marks in a row:
-> Horizontally
-> Vertically
-> Diagonally

If all 9 cells are filled and no player wins, the game ends in a draw.

⚙️ Features

* Two-player gameplay
* 3×3 board display
* Input validation
* Winner detection
* Draw detection
* Simple console interface

C Programming Language
